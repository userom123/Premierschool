# Premier Schools Exhibition Landing Page

A responsive landing page developed from a Figma design using **HTML5, CSS3, and Vanilla JavaScript**. The project replicates the provided UI while following modern frontend development standards, accessibility best practices, and responsive design principles.

## 🚀 Features

- Pixel-perfect Figma to HTML conversion
- Responsive design for Desktop, Tablet, and Mobile
- Semantic HTML5 structure
- BEM naming convention
- Custom CSS (No Bootstrap or Tailwind)
- Hero section with interactive image slider
- Continuous school logo marquee animation
- Mobile-friendly card slider with pagination
- Exhibition highlights slider
- Accessible navigation and forms
- Keyboard navigation support
- Skip-to-content link
- WCAG 2.2 AA accessibility practices
- Cross-browser compatible (Chrome, Firefox, Safari, Edge)

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- CSS Grid
- Flexbox
- CSS Animations
